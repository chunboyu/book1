### UPDATE TO VERSION 4.6.6
------------------------------------------------------------------------
# IMPORTANT NOTES
------------------------------------------------------------------------

1. Save your translation files before overriding with the User Translations

2. Due to unforseen circumstances, Envato have asked us to remove the current Sidebar Plugin that is currently in place. If you update from < 4.4 it may happen, that you have to create your custom sidebars again and assign them with the new given shortcode to the appropriate pages.

3. Also, we had to change the video shortcode syntax from [video] to [embedvideo] if you embedded any videos with the [video] shortcode make sure to change them it to [embedvideo]

4. For all Customizers: If you are using a custom background color / image for the body part you now have to apply it to the #wrapall class

5. [IMPORTANT]: For all users using an older version of the Revolution Slider than 4.2 please update to the latest version. You can find it in the "Extras" folder. There was an important security fix.

------------------------------------------------------------------------
# UPDATE STEPS
------------------------------------------------------------------------

1) Just copy & paste the new "inovado" Folder from the Themes folder you've downloaded on Themeforest in wp-content/themes/ and completely overwrite the old "inovado" folder.

2) To update the Revolution Slider Plugin, unzip revslider.zip in inovado/framework/plugins/ and copy the folder to your WordPress Installation in wp-content/plugins/ and overwrite the old revslider folder.

3) To update the CF-Post-Formats Plugin, unzip the cf-post-formats.zip  in inovado/framework/plugins/ and copy the folder to your WordPress Installation in wp-content/plugins/ and overwrite the old cf-post-formats folder.

3) Login to your WordPress Backend

4) Go to Appearance > Theme Options and hit "Save All Changes", (maybe you have to reload this site after that, if it gives you an error) just to be sure everything gets in the Database and looks good.

------------------------------------------------------------------------
### UPDATE FROM VERSION < 2.0
------------------------------------------------------------------------

Info: After the Update you need to set your Twitter Settings in Appearance > Theme Options > Social Media or deactivate the Twitter Bar in Appearance > Theme Options > Footer if you don't want to use the Twitter Bar.

If you don't like the minor style changes made in 3.0, just replace the style.css & framework/inc/enqueue.php with your old files.
